# F1 Website

A completed F1 fan site built with React and Vite. The project is styled with a dark Pitlane-inspired theme, sharp edges, and blue accent highlights.

## About this project

This project is a complete front-end experience for an F1 season, including:
- Home landing page with featured content and quick season metrics
- Calendar view for the full race schedule
- Race detail page for track-specific information
- My Garage page for saved or favorite races
- My Season page for logged race progress

Animations and transitions are preserved throughout the site, supporting a polished interactive UI.

## Completed work

- Fully functional page navigation and routing
- Race card UI with responsive layout and hover states
- Sprint card styling with blue accents on hover
- Favorite and watched race interactions
- Blurred grayscale background imagery on page sections
- Design token system with dark theme colors, spacing, and typography
- GitHub-safe repository setup with private config excluded

## Features

- React + Vite frontend
- CSS Modules for scoped styling
- Custom token-based design system
- Animated page transitions using GSAP and Flip patterns
- Race calendar grid and detailed race cards
- Dark, modern interface with blue accent highlights

## Project Structure

- `src/`
  - `App.jsx` — application shell, layout, and routing
  - `main.jsx` — React app entry point
  - `components/` — shared UI, layout, and section components
  - `pages/` — page views: Home, Calendar, Race Detail, My Garage, My Season
  - `styles/` — global styling and design tokens
  - `data/` — static race schedule and metadata
  - `assets/images/` — project images and wallpapers
